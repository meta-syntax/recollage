# Recollage — 学習ナレッジWebアプリ

「自分の脳みそを覗ける場所」——学び・体験・思い出・興味を、未来の自分が見返したくなる形で保存し、引き出せるようにする個人開発Webアプリ。

名前は recollect（想起する／集め直す）× collage の造語。ビジュアルカードが並ぶフィード＝自分の記憶のコラージュ。

エンジニアリング事業部のプロジェクト。技術ブランディング（ブログ記事・ポートフォリオへの転用）も視野に入れる。

## コンセプト要約

- 主戦場はホームフィード。「自分の脳みその雑誌」を開く体験
- フィードは「出会い直し」特化（新着＋忘れかけ＋関連の編集的混合）。探す行為は検索・カテゴリツリーに委ねる
- カードは可変密度（構造化の度合いでサイズ・情報量が変わる）× 可変マソンリーレイアウト
- Obsidianインポートでコールドスタートを回避

## 技術スタック（想定）

- フロントエンド: Nuxt
- DB: Supabase 有力（pgvector で関連表示・RAGを実装する想定）。選定はフィード検証後
- ビジュアライズ: 画像生成API + Mermaid
